Dynamic Smart Inventory Management

Problem:

You are tasked with creating a more advanced inventory system for a fictional electronics store. This problem will challenge you to dynamically manage inventory using variables, loops, and conditionals, while handling JavaScript syntax, statements, comments, and variables effectively.

Instructions:

  1. Store Details:

    Declare three variables to store the following details:

    • storeName (string): The name of the store.
    • storeLocation (string): The location of the store.
    • storeCapacity (number): The maximum number of products the store can hold.
  2. Dynamic Product Inventory:

    Create a variable products that stores an array of objects, representing products with:

    • name (string): Name of the product (e.g., Laptop, Smartphone).
    • price (number): Price of the product.
    • quantity (number): Quantity of the product in stock.

    Product 1: Laptop, Price: 18999, Quantity: 50

    Product 2: Smartphone, Price: 9999, Quantity: 100

    Product 3: Tablet, Price: 12999, Quantity: 80

  3. Inventory Validation:

    Write a function checkInventoryCapacity() to check if the total number of products exceeds the store's capacity. If it does, print a warning message.

  4. Product Manipulation (Add & Remove Products Dynamically):

    Write a function addProduct(productName, price, quantity) that adds a new product if there's space. Print a warning if it exceeds the capacity.

    Write another function removeProduct(productName, quantity) that removes a specific quantity of a product. Print an error message if the quantity goes below 0.

  5. Most Expensive Product:

    Write a function getMostExpensiveProduct() to find and return the product with the highest price.

  6. Total Inventory Value:

    Write a function calculateTotalInventoryValue() that returns the total value of the inventory.

  7. User Interaction:

    Use prompt() to add and remove products from the inventory dynamically, and then print the updated inventory value.

  8. Comments:

    Add comments to explain different parts of your code, especially where conditionals and loops are used.

Output:

Store Name: Tech Haven

Store Location: Metro Manila

Total Number of Products: 230

Total Inventory Value: 3649700

Most Expensive Product: Laptop

Store is at full capacity, cannot add new products. Updated Laptop Quantity: 60

Stretch Goal:

Automatic Restocking: Write a function restockProduct(productName, threshold) that adds 20 units if a product falls below 10 units.

Handle product removal edge cases where more than available quantity is removed.

Additional Constraints:

  • Use const and let appropriately.
  • Use loops (e.g., for, forEach) to iterate through products.
  • Use if, else for handling edge cases.
  • Use meaningful variable names and follow JavaScript conventions.
